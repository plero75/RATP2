# Dashboard VHP — Paris-Vincennes

Dashboard temps réel pour l'Hippodrome de Paris-Vincennes : horaires transport, météo, Vélib', trafic et actualités.

## Prérequis

- Node.js

## Installation

```bash
npm install
```

## Lancer en local

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Aperçu du build

```bash
npm run preview
```
